# 🛒 E-commerce Full Stack

Aplicación web de tienda online desarrollada con React y FastAPI.

## 🚀 Tecnologías

* React (Frontend)
* FastAPI (Backend)
* SQLite
* Netlify (Frontend Deploy)
* Render (Backend Deploy)

## 🌐 Demo

https://phenomenal-kitsune-605cf5.netlify.app

## 🔗 API

https://api-productos-p7k5.onrender.com/docs

## ⚙️ Funcionalidades

* Registro y login de usuarios
* Roles (admin / usuario)
* CRUD de productos y categorías
* Carrito de compras
* Finalizar compra

## 📦 Instalación

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
